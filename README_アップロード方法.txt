【v29 変更点】
・原材料ごとに「主材料 / 調味料」を選べる区分列を追加しました。
・主材料の割合を変更しても、調味料の数量kgは固定されます。
・割合合計は主材料だけで100%として計算します。
・調味料は数量kgを直接入力します。
・既存のバックアップや端末保存データは自動的に主材料として読み込まれます。
